⚙️ NPM

🚀 Olá, Bootstrap!

Este site está usando Bootstrap via NPM

📦 package.json
{
"dependencies": {
"bootstrap": "^5.3.0"
}
}
📁 node_modules

Pasta com:

  • bootstrap/
  • @popperjs/
  • (e outras dependências)
🔄 Versões

Fácil de atualizar:

npm update bootstrap

Fácil de compartilhar:

npm install (recria tudo)
🔧 Em projetos reais:

Usamos bundlers como Webpack, Vite ou Parcel para copiar os arquivos d o node_modules para a pasta public automaticamente.

Mas para este exemplo, copiamos manualmente os arquivos d o Bootstrap.
